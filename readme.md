# TINMAN #


## Environment(system) variables ##
## Envirinoment variables should be sent in an .env file at the root of the project ##

Variable  | Value
------------- | -------------
**TINMAN_PORT**  | **5444 or something**
**TINMAN_MODE**  | **dev**
**TINMAN_LOG_PATH**  | **/path/to/log/file**
